# Building-automated-routes

# На данный момент ключи от 2Gis и OpenTripMap не действительны!!!

Веб-приложение для построения пеших маршрутов по городу с учетом интересов пользователя.  
Сервис помогает проложить путь из точки A в точку B через ближайшие достопримечательности, не игнорируя реальную уличную сеть города.

## О проекте

Идея проекта заключается в том, чтобы сделать городские прогулки более содержательными и удобными.  
Пользователь выбирает начальную и конечную точки маршрута, указывает интересующие категории достопримечательностей и допустимое отклонение от основного пути, после чего приложение формирует маршрут через подходящие объекты.

В отличие от обычных навигаторов, приложение не просто ведет из одной точки в другую, а дополняет маршрут интересными местами по пути.  
В отличие от готовых экскурсионных сервисов, пользователь сам влияет на результат: выбирает типы достопримечательностей и управляет маршрутом.

## Основные возможности

— отображение интерактивной карты;
— построение пешего маршрута между двумя точками;
— поиск достопримечательностей по данным OpenTripMap;
— фильтрация объектов по интересам пользователя;
— отбор достопримечательностей по их удаленности от основного маршрута;
— добавление контрольных точек в маршрут;
— удаление маршрута;
— отображение маркеров и названий объектов на карте.

## Как работает

1. Пользователь задает стартовую и конечную точки.
2. Приложение получает список достопримечательностей через OpenTripMap API.
3. Для каждой точки рассчитывается ее положение относительно основного маршрута.
4. В маршрут включаются только те объекты, которые удовлетворяют выбранным условиям.
5. Контрольные точки сортируются, после чего строится итоговый пеший маршрут средствами 2GIS.

## Установка и запуск

Проект представляет собой статическое веб-приложение: страница `index.html` напрямую подключает локальные файлы `style.css`, `utils.js`, `script.js`, а также внешние библиотеки 2GIS MapGL, 2GIS Directions, 2GIS Ruler и Bootstrap через CDN. :contentReference[oaicite:0]{index=0}

1. Клонируйте репозиторий:
git clone https://github.com/username/landmark-web.git
cd landmark-web

2. Запустите локальный сервер. Проект лучше открывать не двойным кликом по index.html, а через HTTP-сервер.
Пример для Python:
   python -m http.server 8000
