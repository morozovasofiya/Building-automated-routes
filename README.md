# Building-automated-routes
Веб-приложение для построения пеших маршрутов по городу с учетом интересов пользователя. Сервис прокладывает путь между двумя точками через достопримечательности, используя данные 2GIS и OpenTripMap, и отбирает объекты так, чтобы маршрут оставался удобным, логичным и не сильно отклонялся от основного маршрута
